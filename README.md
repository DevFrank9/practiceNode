# Wetube Reloaded

- 예외적인 규칙의 라우터 이름 형식

1. / -> HOME
2. /join -> 회원가입
3. /login -> 로그인
4. /search -> 검색

- 보통 규칙의 라우터 이름 형식

1. /users/edit -> 유저 정보 수정
2. /users/delete -> 유저 정보 삭제

3. /videos/watch -> 동영상 보기
4. /videos/edit -> 동영상 편집
5. /videos/delete -> 동영상 삭제
6. /videos/comments -> 동영상 댓글 달기
7. /videos/comments/delete -> 동영상 댓글 삭제
