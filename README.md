# Wetube Reloaded

## 예외적인 규칙의 라우터 이름 형식

1. / -> HOME
2. /join -> 회원가입
3. /login -> 로그인
4. /search -> 검색

## 보통 규칙의 라우터 이름 형식

- 유저

1. /users/:id -> 유저 정보
2. /users/logout -> 유저 로그아웃
3. /users/:id/edit -> 유저 정보 수정
4. /users/delete -> 유저 정보 삭제

- 비디오 화면

1. /videos/:id -> 동영상 보기
2. /videos/:id/edit -> 동영상 편집
3. /videos/:id/delete -> 동영상 삭제
4. /videos/upload -> 동영상 업로드
